## Телеграм-бот для салона красоты

Простой бот на Python/aiogram для салона красоты с двумя основными частями:

- **Прайс-лист**: категории услуг, описание, цена, время.
- **Админка**: добавление/редактирование услуг, просмотр записей.

### 1. Установка

1. Установите **Python 3.10+** (если ещё не установлен).
2. Откройте папку проекта `шаблон бота` в терминале (PowerShell / CMD).
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

(При необходимости используйте `python -m pip install -r requirements.txt` или `py -m pip install -r requirements.txt`.)

### 2. Настройка бота

1. Создайте бота у `@BotFather` и получите **токен**.
2. Откройте файл `config.py` и укажите:

- **BOT_TOKEN** — токен вашего бота;
- **ADMIN_IDS** — список Telegram ID администраторов, которые будут иметь доступ к админке.

Пример:

```python
BOT_TOKEN = "1234567890:ABCDEF..."
ADMIN_IDS = [111111111, 222222222]
```

Узнать свой Telegram ID можно через ботов `@userinfobot` или `@myidbot`.

### 3. Запуск бота

В терминале в папке проекта выполните:

```bash
python bot.py
```

Если используется `py`:

```bash
py bot.py
```

После запуска найдите бота в Telegram и нажмите **Start**.

### 4. Функционал для клиентов

- **Команда** `/start` — приветствие и главное меню.
- Кнопки:
  - **«Прайс-лист 💅»** — выбор категории, затем списка услуг с:
    - названием,
    - описанием,
    - ценой,
    - временем выполнения;
    можно открыть услугу и увидеть кнопку «Записаться».
  - **«Записаться 🗓»** — быстрый переход к выбору услуги и записи.
- При записи бот спрашивает:
  - услугу;
  - желаемую дату и время (формат `ДД.ММ.ГГГГ ЧЧ:ММ`);
  - имя и фамилию;
  - номер телефона.

Заявка сохраняется в базе и отправляется администратору.

### 5. Админ-панель

Вход в админку — по команде:

- **`/admin`** — доступ есть только у пользователей с ID из `ADMIN_IDS`.

После входа появляются кнопки:

- **«📋 Услуги»** — показать текущий прайс-лист (ID, категория, название, описание, цена, время).
- **«➕ Добавить услугу»** — по шагам:
  - категория (например, Маникюр, Педикюр, Уход за кожей);
  - название;
  - описание;
  - цена;
  - время выполнения (минуты).
- **«💰 Обновить цену»**:
  - бот показывает список услуг с ID и ценами;
  - админ вводит ID услуги и новую цену.
- **«⏱ Обновить время»**:
  - бот показывает список услуг с ID и временем;
  - админ вводит ID услуги и новое время в минутах.
- **«📆 Записи»** — список последних заявок клиентов (услуга, дата/время, имя, телефон, Telegram, статус).
- **«⬅ Назад в главное меню»** — возврат к клиентскому меню.

Таким образом:

- **Прайс-лист** редактируется через добавление услуг и изменение цены/времени.
- **Записи** хранятся в SQLite-базе `beauty_salon.db` в корне проекта.

Бот спроектирован так, чтобы быть:

- простым для клиентов (кнопки, пошаговые вопросы);
- удобным для администратора (админ-меню, управление прайсом и просмотр заявок).
